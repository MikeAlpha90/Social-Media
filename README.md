# Social-Media WebApp

## Description:
This is a social media web app built using the MERN stack (MongoDB, Express, React, Node.js). It’s designed to mimic a Facebook-like layout, with a topbar, left sidebar, center feed for posts, and a right sidebar featuring ads, birthdays, and online friends.

### Features:
Topbar: Displays the app name and options for navigation.
Left Sidebar: Contains sections for feeds, chats, video groups, and a friend list.
Center Feed: Displays all posts from users, along with a "Write a Post" option.
Right Sidebar: Shows ads, birthdays, and online friends.
This project integrates frontend (React), backend (Node.js and Express), and a MongoDB database for storing user data and posts. The app allows users to interact with the feed, write posts, and see what their friends are doing.

The app’s API handles all the communication between the frontend and the backend, managing posts, friends, and user profiles.

If you want to explore this project further, you can try running it locally and testing out all its features.
